# Grid Start Shop - Teste Técnico PHP

Projeto desenvolvido como teste técnico para vaga de estágio.

## Funcionalidades

- Página inicial de loja virtual
- Vitrine de produtos
- Sistema de cadastro de usuário
- Sistema de login
- Área restrita (dashboard)

## Tecnologias utilizadas

- PHP
- MySQL
- HTML
- CSS
- Bootstrap

## Como executar o projeto

1. Clone o repositório
2. Configure um servidor local (XAMPP, WAMP ou Laragon)
3. Importe o banco de dados
4. Execute o projeto no navegador
